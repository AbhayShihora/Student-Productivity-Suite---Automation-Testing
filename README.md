# 🚀 Student Productivity Suite - Automation Testing

## 📌 Project Overview

This project demonstrates automation testing of a JavaScript-based web application using Selenium WebDriver.

The application tested is a **Student Productivity Suite**, where users can register and log in to manage their productivity tasks.

---

## 🛠️ Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven

---

## ⚙️ Framework Design

* Page Object Model (POM)
* Structured test framework
* Reusable components

---

## ✅ Features Automated

* User Registration
* User Login
* Validation of success/error messages

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone https://github.com/your-username/student-productivity-automation.git

2. Open in Eclipse or IntelliJ IDEA

3. Update Maven dependencies

4. Run test:
   Right click on test class → Run As → TestNG Test
